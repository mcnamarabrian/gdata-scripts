gdata-scripts by mcnamarabrian

Miscellaneous Python scripts for administering Google Apps

Google exposes a great deal of administrative functionality beyond the standard Google Apps control panel via API.

Dependencies

The Python scripts in this repository will work with the gdata-2.0.10 library and should work with Python 2.4.x - 2.7.x.
